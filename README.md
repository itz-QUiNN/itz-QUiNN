# Hi there, I'm Ishara Samarathunga 👋

**Python Developer | Data Extraction Specialist | AI Enthusiast**

Welcome to my GitHub! I specialize in building high-performance web scrapers, robust data pipelines, and automated workflows. With over two years of hands-on experience, my focus is on engineering resilient systems capable of navigating complex anti-bot environments to deliver clean, actionable data. 

Currently, I'm pursuing a Bachelor's degree in **Artificial Intelligence**, with the goal of bridging the gap between raw data engineering and advanced machine learning capabilities.

### 🚀 What I Do Best

* **Web Scraping & Automation:** Crafting stealthy, scalable scrapers using Playwright, Selenium, DrissionPage, and Scrapy.
* **Anti-Bot Engineering:** Strategically bypassing advanced protections like Cloudflare, PerimeterX, and Akamai.
* **Data Pipelines:** Designing efficient end-to-end ETL processes that turn unstructured web data into structured insights.
* **Modern & Fast Tooling:** Passionate about optimizing development environments for speed and productivity, heavily utilizing modern package managers like `uv` alongside heavily customized VS Code setups.
